Livewell Tools

Internal operational tools for multi-location retail inventory.

Tools:
• Bin Lookup – warehouse kiosk for bin location
• Transfers – builds inter-store replenishment picks
• Scan + Putaway – receiving, putaway, and Shopify update staging

Workflow:
1. Update pickle-data.csv in the repo
2. Tools auto-load latest data
3. CSV outputs are uploaded back into Shopify
